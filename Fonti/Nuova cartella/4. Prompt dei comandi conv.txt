Guida all’utilizzo del prompt dei comandi (MacOS)
Guida ai comandi utili per il corso Il prompt dei comandi nel Mac si chiama Terminal. Sii apre scrivendo Terminal
nello spotlight oppure scegliendo Terminal nelle appl cazioni.


Il principio di funzionamento è lo stesso del prompt dei comandi di Windows, i
comandi sono leggermente differenti, in particolare:


• il comando cd funziona nella stessa maniera,
• il comando dir è rimpiazzato da ls (scrivere ls -l per una bella visualizzazione) • il comando gcc si usa nella stessa maniera, in particolare gcc -o
<nomeprogramma> <nomefile.c> permette di compilare il file dal nome

nomefile.c nella cartella corrente ed ottenere un file eseguibile dal nome
nomeprogramma

• Se non si specifica la parte -o <nomeprogramma> verrà creato un eseguibile dal nome
a.out.
